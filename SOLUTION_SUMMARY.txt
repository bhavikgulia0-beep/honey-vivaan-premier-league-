Option 1: NETLIFY (EASIEST - 2 minutes)
────────────────────────────────────────
1. git push origin main
2. Go to netlify.com
3. Click "New site from Git"
4. Select your repo
5. Auto-deploys! 🎉

Option 2: VERCEL (2 minutes)
────────────────────────────────────────
1. Go to vercel.com
2. Import GitHub repo
3. Click Deploy
4. Done!

Option 3: GITHUB PAGES (2 minutes, completely free)
────────────────────────────────────────
1. git push origin main
2. Go to repo Settings → Pages
3. Select main branch
4. Published at: https://YOUR-USERNAME.github.io/repo-name

Option 4: REPLIT (1 minute, no git needed)
────────────────────────────────────────
1. Go to replit.com
2. Click Create → Import from GitHub
3. Paste your repo URL
4. Auto-deploys with public URL!

Option 5: LOCAL TESTING (Instant)
────────────────────────────────────────
python -m http.server 8000
Then open http://localhost:8000

🎮 GAME RULES:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Two players pick number 1-6 each turn
2. Same number = OUT (switch batting roles)
3. Different number = runs scored
4. After both bat once = highest score wins
5. Simple, fun, quick!
